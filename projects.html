Diabetes Prediction System

An advanced machine learning system that predicts diabetes risk with 78% accuracy using:

  • Python with Scikit-learn, Pandas and NumPy
  • Random Forest Classifier algorithm
  • Feature importance analysis to identify key health indicators
  • Interactive web interface built with Flask
  • Data preprocessing pipeline for handling missing values
  • Model evaluation using precision-recall metrics

The system analyzes 8 key health indicators including glucose levels, BMI, and age to provide personalized risk assessments.
